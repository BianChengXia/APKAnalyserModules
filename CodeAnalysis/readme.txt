模块功能：

使用Decompile模块生成的java源文件进行分析
编写一定的代码匹配规则，匹配出java代码中调用的API（40个），敏感代码（24）个，以及代码中存在的URL，Email等
将数据保存在全局变量中：
+ extern NORMALs	ul;
+ extern NORMALs	em;
+ extern APITracing	api_t;
+ extern ReviewTracing	rev_t;

+ extern bool		native;
+ extern bool		dynamic;
+ extern bool		reflect;
+ extern bool		crypto;
+ extern bool		obfus;
